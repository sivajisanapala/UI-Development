OOPs Basics

Abstraction: It is a feature of OOPS where we show the essentials and hide the details of any metod.

Encapsulation: It is a feature of OOPS where we group all related Functions/methods and variables/data into an object. Like an object with complate details of a person.

Inheritance: It is a feature of OOPS where we use same method from base/parent class in Derived class. In JavaScript/TypeScript it is supported be Prototype.

Polymorphism: It is a feature of OOPS where we use same method/function for different Objects.
